# 마크다운의 장점
- 간단한 문법
- 마크다운에서 html간의 일대일 관계 존재 (대응 X)

# 마크다운의 장점
- 결과물에 대한 미세 조정이 불가능

# 가능한 표현들
- header -> `# 수로 h1에서 h6까지 표현가능 예) # 큰 제목`
- text decoration -> 볼드, 이탤릭
- list 표현 -> 들여쓰면 목록 in 목록 가능
- 이미지 첨부 -> `[단어](http://링크`
- 인용 가능
- ``` ` 하나면 일반 코드 ` ` `{언어} 면 multiline 코드 및 syntax highlighting ```
